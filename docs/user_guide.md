# Руководство пользователя MOznoDS

MOznoDS — это веб-приложение для групповых голосовых звонков и обмена сообщениями, упрощенный аналог Discord.

## 1. Регистрация и Вход

Для доступа к функциям приложения необходимо авторизоваться.

*   **Регистрация**: Требуется специальный **код-приглашение** (Invite Code), который выдается администратором.
*   **Профиль**: После регистрации создается профиль пользователя, где можно указать отображаемое имя (`display_name`).

## 2. Комнаты (Rooms)

Основное общение происходит в комнатах.

*   **Создание комнаты**: Любой пользователь может создать новую комнату, указав её название. Создатель становится **владельцем** (Owner).
*   **Управление**: Владелец может изменять название комнаты или удалить её.
*   **Участие**: Пользователи могут присоединяться к существующим комнатам.
*   **Список комнат**: Вы видите только те комнаты, в которых вы являетесь участником.

## 3. Текстовый чат

В каждой комнате есть текстовый чат.

*   **Сообщения**: Вы можете отправлять текстовые сообщения, которые мгновенно доставляются всем участникам комнаты (через WebSocket).
*   **История**: Доступна история сообщений с пагинацией.
*   **Вложения**: К сообщениям можно прикреплять файлы.

## 4. Файлы

Вы можете загружать файлы и делиться ими в чате.

*   **Лимиты**: Максимальный размер файла — **10 МБ**.
*   **Типы файлов**: Поддерживаются изображения (`image/*`), PDF (`application/pdf`) и текстовые файлы (`text/*`).
*   **Хранение**: Файлы привязываются к вашему пользователю.

## 5. Голосовые звонки (WebRTC)

Приложение поддерживает голосовые звонки внутри комнат.

*   **Подключение**: Вы можете "войти" в голосовой канал комнаты.
*   **Технология**: Используется WebRTC (P2P Mesh). Это значит, что данные (голос/видео) передаются напрямую между пользователями, минуя сервер (сервер используется только для соединения/сигнализации).
*   **Статусы**: Приложение отображает, кто сейчас находится в звонке.

## 6. Технические особенности

*   **Real-time**: Чаты и звонки работают в реальном времени.
*   **Безопасность**: Доступ к комнатам и файлам имеют только авторизованные участники.
