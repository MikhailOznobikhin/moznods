# План этапа 2 (MOznoDS)

Этот документ описывает задачи и прогресс по второму этапу разработки платформы MOznoDS.

## Задачи

1. **[x] Исправить пропадание звука при сворачивании звонка**
   - Проблема: `VideoPlayer` размонтируется при `isExpanded: false`.
   - Решение: Оставлять компоненты смонтированными, но скрытыми (через CSS `hidden`).

2. **[x] Изменить "You" на "Ник пользователя (You)" в голосовых звонках**
   - Проблема: Локальный пользователь отображается просто как "You".
   - Решение: Используется имя пользователя из `useAuthStore`.

3. **[x] Добавить демонстрацию экрана**
   - Добавлена кнопка в `CallOverlay`.
   - Реализован захват потока через `getDisplayMedia` и замена трека в `useCallStore`.

4. **[x] Обновить заголовок и иконку на главной странице**
   - Изменен `<title>` в `index.html`.

5. **[x] Браузерные уведомления**
   - Добавлен запрос разрешений.
   - Реализовано отслеживание `visibilitychange`.
   - Добавлены настройки уведомлений в `SettingsPage`.

6. **[x] Индикация прочитанных/непрочитанных сообщений**
   - Backend: Добавлено поле `read_by` в `Message`.
   - Frontend: Отображение галочек (одна - отправлено, две синие - прочитано кем-то кроме автора).

7. **[x] Поделиться ссылкой на комнату**
   - Backend: Модель `RoomInvitation`, сервис и эндпоинты.
   - Frontend: `ShareRoomModal` для генерации и `InvitePage` для входа.
   - Поддержка входа после регистрации (через `localStorage`).

8. **[x] Кликабельные ссылки в чате**
   - Реализована функция `linkify` в `MessageList`.

9. **[x] Полноэкранный режим для видео/стримов**
   - Добавлена кнопка развертывания в `VideoPlayer`.
   - Использование Fullscreen API для погружения в просмотр.
