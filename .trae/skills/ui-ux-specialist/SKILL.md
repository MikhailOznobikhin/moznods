# Специалист по UI/UX

Этот навык предназначен для улучшения пользовательского опыта (UX) и визуального интерфейса (UI) проекта MOznoDS.

## Принципы дизайна

1. **Консистентность**: Следовать существующей цветовой схеме (Slate/Gray 900 для фона, Blue 600 для акцентов).
2. **Доступность (A11y)**:
   - Проверять контрастность текста.
   - Добавлять `aria-label` для иконок без текста.
   - Обеспечивать навигацию с клавиатуры.
3. **Отзывчивость (Responsive)**: Использовать брейкпоинты Tailwind (`sm:`, `md:`, `lg:`) для корректного отображения на мобильных устройствах.
4. **Визуальная иерархия**: Использовать размер шрифта, вес и цвет для выделения важных элементов.

## Лучшие практики для MOznoDS

- **Чат**: Сообщения должны иметь четкие границы или отступы. Индикаторы статуса (прочитано/отправлено) должны быть заметны, но ненавязчивы.
- **Звонки**: Элементы управления звонком должны быть легко доступны одной рукой на мобильных устройствах.
- **Темная тема**: Сохранять глубокий темный фон для снижения нагрузки на глаза, характерный для Discord-подобных приложений.

## Когда использовать этот навык

- Перед созданием новых сложных UI-компонентов.
- При получении жалоб от пользователей на неудобство интерфейса.
- Для проведения аудита доступности (accessibility audit).
- Когда нужно добавить "лоск" и профессиональный вид существующим страницам.
